四月计划

发表于 2017-03-29   |   分类于 计划

四月计划清单

C++语言相关

  1. 《C++ primer》需重看一遍，结合《effective C++》
  2. 《深入探索C++对象模型也需要看》
  3. 《STL源码剖析》
  4. 能完成STL源码的简单实现（若时间不够可每周完成一个容器）

数据结构和算法

  1. 刷leetcode上的题目
  2. 《剑指offer》，《程序员代码面试指南等书》
  3. 基本知识点的掌握（红黑树，排序等）

基础

  1. 操作系统（内核，线程池，汤子瀛的书）
  2. 计算机网络（tcp/ip详解卷1，谢希仁的书）

linux环境编程

  1. 《unix高级环境编程》
  2. 《unix网络编程》
  3. 《多线程服务端开发》
  4. 《linux/unix系统编程手册》
  5. 《unix/linux编程实践教程》
  6. 在上述书中选合适的开始看，起码四月份应该对这方面知识点有大致了解
  7. 学习使用linux（基本shell指令等）

项目相关

  1. 写好STL源码（四月便开始）
  2. 看了linux环境编程初期可尝试做web服务器等
  3. 阅读redis源码

大致规划

  1. 四五月份重点（C++语言，刷题，网络编程多线程编程相关，后期进入状态有精力争取开始读源码，抓项目）
    - C++语言基础（effective，primer，object model）
- 实现STL源码（最好一周一个part）
- 保持刷题（hihocoder，leetcode，剑指offer等）
- 学习linux下编程以及网络编程(15号前争取尽量读完unix/linux编程实践教程)
- 仔细阅读tcp/ip详解，操作系统等
- 后期相关书籍（unix网络编程)
- 可以读实际源码（redis，muduo，加油争取！）
- 今早明确要做的项目（课程项目，网络编程相关）
- 有空可以看看python脚本语言，大致了解
- 时不时刷牛客（笔试题等）和参加hihocoder比赛等，保持警惕！早睡早起
  2. 六月份开始抓紧做项目（四五月要确立项目内容），同时阅读源码，复习C++，仔细看网络和操作系统。
  3. 四月每周安排：
    <font color=red>- 每天保持刷三题（可分早中晚，注意对基础的关注）
- 上午看C++（15号前尽量刷完effective，结合primer相关章节）
- 下午看unix/linux编程实践教程（结合APUE个别章节，看到一定程度开始看unp）
- 晚上可以写stl源码或者忙项目相关（做小型c++相关项目或者course或者着手准备项目问题）
- 结合相关内容，有时记得看看计算机网络基础相关，操作系统基础相关（tcp/ip详解，现代操作系统）</font>
- 关注有没有合适实习机会
  4. 个人面试问题总结
    - 没有项目，实践经验匮乏，面试尴尬。（重中之重，力求做出至少一个小型能写上简历的项目）
- 基础一般（C++开发所需能力点：C++语言基础并不熟，算法仍然不足以应付笔试面试，基础知识如数据库设计模式操作系统网络等都不太熟不足以应付笔试，c++开发所需技能多线程编程网络编程都不会。）
- 简历：没有足够的项目（aim：独立开发的c++/c小型项目+course（可选）+...），专业技能一栏匮乏（aim：多多电亮技能树，C/C++熟悉精通，熟读STL源码（模拟写过），精通常用数据结构算法，熟悉linux系统和其常用命令及脚本编程（shell，gcc，gdb，make），熟悉linux系统编程，网络编程，若能阅读完redis源码了解python写上更佳）
- 保证基础！努力增加项目！努力增加面试可聊点！针对项目各种问题努力准备！面试中详细表述！后期准备面试各个常问的概括性问题（如何证明自己的技术能力等）。
- 加油！加油！
- offer = 项目+基础+语言表达+自信！准备完善才能自信！
- 总之：项目 + C++ + linux + 网络编程/多线程编程 + 各个经典算法题数据结构 + 操作系统计算机网络等基础知识 = C/C++研发！目标秋招！加油！
